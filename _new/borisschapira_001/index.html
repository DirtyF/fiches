Construire une grille fluide

Principes abordés : Responsive Web Design

Niveau requis : bases en HML, en CSS

Acte 1

Nous allons construire ensemble une grille fluide... Un élément de type bloc est aussi large que son parent. Imaginons qu'il est "large à 100%", il sera la base de notre grille.

<div class="grille">
    <!-- 100% large -->
</div>
Mettez en place deux colonnes de contenu. Le contenu principal occupe 2/3 de la largeur disponible, le contenu latéral, 1/3. 
<div class="grille">
    <div class="col-2-3">
        Contenu principal
    </div>
    <div class="col-1-3">
        Contenu latéral
    </div>
</div>

Pour faire en sorte d'afficher les deux zones l'une à côté de l'autre, il faut les faire flotter et leur appliquer des largeurs...

Proposition de solution :

[class*='col-'] {
	float: left;
}

.col-2-3 {
	width: 66.66%;
}

.col-1-3 {
    width: 33.33%;
}

Acte 3

Le plus difficile dans la mise en place d'une grille fluide, c'est de mettre en place les gouttières. Jusqu'ici nous avons réussi à rendre notre grille flexible, et nous pourrions continuer de même pour les gouttières, avec des calculs mathématiques de pourcentages. Mais la plupart du temps, le rendu est décevant, avec des gouttières qui laissent apparaitre trop d'espaces entre les contenus. Faisons simple, mettons en place des gouttières de taille fixe.

Pour simplifier la mise en place, nous allons utiliser la règle box-sizing: border-box; afin que la largeur définie d'un contenu soit véritablement sa largeur, indépendamment des padding et des border appliqués.

Mettez en place box-sizing: border-box; pour l'ensemble des éléments, puis définissez des gouttières de 20 pixels.

Essayez d'imaginer quels impacts pourrait avoir l'absence de box-sizing: border-box;. De plus, que faut-il faire pour que cela fonctionne sur les navigateurs principaux ?

Attention, les gouttières ne sont qu'entre les contenus... ni à gauche, ni à droite de l'ensemble.

Proposition de solution pour la mise en place de box-sizing: border-box;:

*, *:after, *:before {
	-webkit-box-sizing: border-box;
	-moz-box-sizing: border-box;
	box-sizing: border-box;
}

Puis pour la mise en place des gouttières :

[class*='col-'] {
	padding-right: 20px;
}
[class*='col-']:last-of-type {
padding-right: 0;
	}

Acte 4

Nous voudrions pouvoir nous laisser une option sur les gouttières extérieures, sous la forme d'une classe CSS supplémentaire. Implémentez cette classe .grille-espace. 
<div class="grille grille-espace">
	<div class="col-2-3">
		Contenu principal
	</div>
	<div class="col-1-3">
		Contenu latéral
	</div>
</div>

Ajoutons d'abord le padding à gauche :

.grille-espace {
	padding-left: 20px;
}

Puis nous restorons éventuellement la gouttière de droite que nous avons supprimé précédemment :

.grille-espace > [class*='col-']:last-of-type {
	padding-right: 20px;
}

Acte 5

Notre grille est bien partie, mais il faut désormais ajouter d'autres types de largeurs d'éléments pour rendre notre mise-en-page flexible aux besoins.

Ajoutez des classes pour mettre en forme des contenu occupant 1/2, 1/4 ou 1/6 de la largeur disponible.

Proposition de solution :

.col-1-2 {
	width: 50%;
}
.col-1-4 {
	width: 25%;
}
.col-1-6 {
	width: 16,66%;
}

Acte 6

Notre grille est prête, mais résistera-t-elle au temps ? Pas si sûr...

Que se passerait-il demain si nous voulions modifier la largeur de nos gouttières ? Proposez une solution pour rendre la maintenance plus simple. Quels impacts sur le processus de développement ?

Il existe, dans le monde du développement CSS, la notion de pré ou de post-processeurs. Peuvent-ils aider ? Comment ? Est-ce lourd à mettre en place ?

Proposition de solution en SCSS :

/* Application de border-box */
* {
	@include box-sizing(border-box);
}

/* On définit les gouttières dans un paramètre facilement modifiable */
$gouttiere: 20px;

.grille {
	background: white;
	margin: 0 0 $gouttiere 0;

	/* Ici quelques règles pour gérer le contenu situé après la grille */
	&:after {
		/* Ou @extend clearfix */
		content: "";
		display: table;
		clear: both;
	}

	[class*='col-'] {
		float: left;
		padding-right: $gouttiere;

		.grille &:last-of-type {
			padding-right: 0;
		}
	}

	.col-2-3 {
		width: 66.66%;
	}
	.col-1-3 {
		width: 33.33%;
	}
	.col-1-2 {
		width: 50%;
	}
	.col-1-4 {
		width: 25%;
	}
	.col-1-6 {
		width: 16.66%;
	}

	/* Option pour gouttières extérieures */
	.grille-espace {
		padding-left: $gouttiere;

		[class*='col-']:last-of-type {
			padding-right: $gouttiere;
		}
	}
}
